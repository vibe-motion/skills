# Place Aliases

Use this alias map before calling the exporter.

## Japan Mainland (No Hokkaido preset)

- `日本`
- `Japan`
- `日本本岛`
- `日本本州`
- `本州`
- `本州岛`
- `日本岛`

These trigger default `--component-centroid-lat-max=42.2`.

## Country Name Aliases

- `印度尼西亚` -> `Indonesia`
- `中国` -> `China`
- `美国` / `USA` -> `United States of America`
- `英国` / `UK` -> `United Kingdom`

If alias is not listed, pass the original place string as `--country-name`.
